# GUI布局更新说明

## 🎯 新的布局结构

运行控制页面现在采用**左右分栏布局**：

### 📋 左侧主控制区
- **目录选择**：本地视频目录配置
- **运行控制**：开始/停止按钮
- **最大翻页**：抓取页数限制
- **页面延时**：抓取间隔设置

### 📊 右侧进度监控区
#### 🔝 总进度与状态
- 进度条显示整体查重进度
- 百分比显示（实时更新）
- 状态信息（就绪/运行中/完成）

#### 📋 三列任务卡片
- **任务1-3**：并行显示多个模特的进度
- 每个卡片显示：
  - 模特名称
  - 进度条
  - 完成百分比

#### 📝 可折叠日志面板
- **默认展开**，可点击"折叠"按钮收起
- 显示查重过程中的详细日志
- "清空"按钮可清除日志内容
- 支持滚动查看历史日志

## ✨ 主要改进

1. **独立监控区域**：进度和日志有专门的显示区域
2. **实时更新**：进度条和百分比实时刷新
3. **多任务支持**：3列并行显示多个任务进度
4. **灵活布局**：日志面板可折叠，节省空间
5. **清晰标识**：各区域有明确的标签和分组

## 🔧 技术实现

- 使用`grid布局`实现左右分栏
- 左侧权重4，右侧权重2
- `progress_var`控制主进度条
- `task_cards`列表管理3个任务卡片
- `log_panel_expanded`控制日志面板展开/收起状态

## 📱 使用说明

1. **启动程序**：查看右侧"进度监控"区域
2. **运行查重**：进度条实时更新，任务卡片显示各模特进度
3. **查看日志**：在日志面板查看详细运行信息
4. **管理空间**：可折叠日志面板获得更多任务卡片显示空间
